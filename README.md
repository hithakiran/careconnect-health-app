# CareConnect – Healthcare Support Dashboard 🏥

## 📌 Overview
CareConnect is a simple healthcare support web application designed for NGOs to collect patient requests and provide instant preliminary guidance using a rule-based AI system.

---

## 🚀 Features

- 🧾 Patient Support Form  
- 🤖 AI-based Symptom Analysis (Triage System)  
- ⚠️ Severity Classification (Low / Medium / High)  
- 📊 Dashboard to View Patient Requests  
- 💾 Local Storage (Data persists after refresh)  
- 💬 Chatbot for Basic Healthcare Assistance  
- 🎨 Clean and Responsive UI  

---

## 🧠 AI / Automation Concept

The application uses a **rule-based triage system** that analyzes user-entered symptoms and:

- Determines severity level (Low / Medium / High)  
- Provides basic health advice  
- Helps prioritize urgent cases  

The chatbot uses the **same logic**, ensuring consistent responses across the platform.
